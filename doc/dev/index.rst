================
 內部開發者文件
================

.. note:: 如果你要找的是如何对接 Ceph 和你自己的软件，那么你应该去看这个文档： \
   :doc:`/api/index` 。

编译完之后，你可以用以下命令启动开发者模式的 Ceph 集群： ::

	cd src
	install -d -m0755 out dev/osd0
	./vstart.sh -n -x -l
	# check that it's there
	./ceph health
